已知问题

首先目前黑苹果分为 Clover 和 OpenCore 两个版本，OC是趋势，但还有很多问题，于是我安装了问题相对较少的 Clover 版本。

根据 GitHub issue，目前 Clover 版本存在以下问题：

  • 有线网在macOS 10.15下可能无法工作
    • 我没有遇到
  • 指纹传感器无法工作
    • 直接禁用
  • 英特尔蓝牙可能会导致睡眠问题，并且不支持部分蓝牙设备
    • 我没有遇到
  • 独立显卡无法工作
    • 黑苹果老问题了，直接禁用
  • 英特尔无线网卡无法工作
    • 同样老问题，需要另买网卡
  • 瑞昱USB SD读卡器无法工作
    • 直接禁用

具体可以上GitHub仓库看。

安装黑苹果

准备工作：

  • 一个8GiB以上容量的U盘，建议USB3.0。
  • 下载黑苹果镜像 @黑果小兵
  • 来自 小米社区@jiajieit 的Windows工具合集 MEGA网盘
  • 在 GitHub releases页面 下载最新的EFI
  • 用磁盘管理压缩一个空白分区用来安装黑苹果，我分了60G。

制作安装盘

用管理员权限打开TransMac。右键点击你要制作的安装盘，选择Restore with Disk Image，选择下载好的镜像，提示格式化磁盘，选择OK。之后只需等待写盘完成。黑果小兵现在的镜像都自带了WEPE，很方便。

写盘完成后，打开DiskGenius，打开安装盘的ESP分区，删除EFI目录。把下载好的EFI解压后拷贝进去。

安装盘制作完成。

添加引导

打开硬盘的ESP分区，把下载好的EFI拷贝进去，此时EFI文件夹中应该多出来了一个CLOVER文件夹。

打开BOOTICE，选择磁盘。点击分区管理，给ESP分区分配一个盘符。关闭分区管理，选择UEFI，添加一个启动项，启动文件选择ESP分区下的\EFI\CLOVER\CLOVERX64.EFI。把CLOVER移到最上面。保存设置。

进入BIOS，在Security 中给BIOS设置一个密码。点击Secure Boot Mode，改为Disabled，关闭安全启动。记得保存设置。

Extra：ESP分区扩容

macOS需要EFI分区容量大于200M，如果容量不足，就需要扩容了。

U盘启动到PE，使用分区工具删除MSR分区，把多出来的空间全部扩容到ESP分区。

安装macOS

U盘启动到macOS安装程序。先使用磁盘工具，把用来安装macOS的分区格式化成APFS格式。退出磁盘工具，开始按步骤安装，可能会重启数次，be patient。

后续工作

最大的问题就是无线网卡驱动了，https://github.com/chris1111/Wireless-USB-Adapter-Clover 这个仓库可以解决绝大多数网卡的驱动问题。

无线网卡的选购我推荐COMFAST-811ac，也就二十多块。
