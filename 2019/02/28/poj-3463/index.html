POJ 3463 Sightseeing

最短路与次短路

POJ3463 Sightseeing

题意：

求图中最短路和比最短路长度大1的路径数总和

思路:

先求出最短路数和次短路数，如果次短路长度 = 最短路长度，那么输出路径数和，否则输出最短路数。 求次短路数也非常简单，在Dijkstra过程中分类讨论即可。

代码:

# 图论, 最短路
POJ 1135 Domino Effect
Luogu P2051 [AHOI2009] 中国象棋
