Luogu P1600 [NOIP2016] 天天爱跑步

qwq

首先有一个对于树上路径问题的很重要思路：把路径 (p, q) 拆成 (p, lca), (lca, q).

接下来考虑每条路径 (s, t) 对于一点x是如何产生贡献的。

两种情况，s 在 x 子树内，t 在 x 子树内。

当 s 在 x 子树内，需满足 $dep_s - dep_x = w_x$。移项：$w_x + dep_x = dep_s$。

当 t 在 x 子树内，需满足 $dep_s + dep_x - 2dep_{lca} = w_x$。移项：$w_x - dep_x = dep_s - 2dep_{lca}$。

也就是说，对于每个子树，我们需要知道有多少点满足 $w_x - dep_x = dep_s - 2dep_{lca}$ 或 $w_x + dep_x = dep_s$。这个可以每个节点开一颗动态开点线段树来维护。对于每条路径，我们需要维护 $dep_s - 2dep_{lca}$ 和 $dep_s$。

对于路径 (s, t)，我们分别在 s 和 t 的线段树维护 $dep_s$ 、$dep_s - 2dep_{lca}$，用树上差分的思想对 $LCA(s, t)$ 和 $fa_{LCA(s, t)}$ 操作。最终一遍 dfs 得到答案。

需要注意的是有可能 $dep_s - 2dep_{lca} \le 0$ ，所以需要整体加一个偏移量。
